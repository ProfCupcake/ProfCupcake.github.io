Arma3Sync Repo Guide

Intro - Connecting - Mod Groups - Extras - Troubleshooting

Before you can actually use mods with Arma3Sync, they need to be added to mod groups. This is a fairly simple process.

On the addons screen, right-click on the right-hand pane, then click "Add Group". Name it something, like "Geoffrey", "Steve", or "2MRB Standard". It'll then appear on the right. To add mods to it, just drag them over from the left and drop them onto it.

To start Arma with the mod group, check the box next to it. You can also expand the group and select individual mods to enable/disable, as well as enable other groups or mods to also run with.

Your 2MRB mod group should, at minimum, contain the set of required mods for us. There is a list of modfolder names in the Discord, in #modlist-reference, that you can use to check everything's there.
